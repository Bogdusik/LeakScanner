# LeakScanner Chrome Extension

Chrome extension for scanning GitHub/GitLab repositories for security vulnerabilities.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Load Extension

1. Build the extension: `npm run build`
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the `dist` folder
